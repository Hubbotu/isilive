This folder is split into tracked examples and private local files.

Tracked:
- `settings.example.json`: shared baseline config for all machines

Local only:
- `settings.json`: your real local Claude/Codex settings
- any notes, session state, or machine-specific files

Recommended setup on each machine:
1. Copy `settings.example.json` to `settings.json`
2. Add any private or machine-specific overrides only in `settings.json`
